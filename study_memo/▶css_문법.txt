▶선택자(h1), 속성(color), 속성값(#f00)

HTML에 적용하는 3가지 방법

1)인라인 스타일
index.html 내에서 직접적으로 스타일 시트로 정의하는것. 거의 안씀
ex) <h1 style="color: #f00">CSS3 Selector Basic</h1>

2)내부 스타일 (DOCTYPE에서 주고 들어가는것)
   <style>
      p {
        font-size: 20px;
      }
    </style>

3)외부 스타일 (*실무-99%사용)
style.css로 파일을 따로 만들어서 index.html과 연동시키는것.
index.html에서 :    <link rel="stylesheet" href="./style2.0.css" />


★CSS3 선택자★
1) 전체선택자
- wildcard 선택자
- reset css
- * (프로그래밍 언어에서 전체, all을 의미, 모든 요소에 정의하겠다는 뜻)

2) 태그선택자
- 특정한 태그를 선택해서 스타일을 적용하고 싶을 때
- EX) h1 Tag, p Tag 등등
(*아이디선택자와 다른점: 태그선택자는 스타일이 모두 적용됨)

3) 아이디선택자 명령어: #
- html 태그 중에서 id 속성값을 가지고 있는 태그들에게 스타일을 적용
-  "#" 
- id 속성값은 반드시 html에서 1번만 사용해야 한다. (나중에 JAVA에서 불러올때 문제가 되기 때문에)

4) 클래스선택자  명령어: . (온점)
- html 태그 중에서 class속성값을 가지고 있는 태그들에게 스타일을 적용
-  "."
- class속성값은 html에서 여러번 횟수제한없이 사용이 가능하다.
- 반복적으로 공통된 스타일을 정의하고 적용하고자 할 때

5) 후손선택자 명령어: 띄어쓰기
- 부모 아래에 들어오는 모든 자식요소들
- 자손선택자도 모두 포함
- "띄어쓰기"

6) 자손선택자 명령어: >
- 부모의 직계 자식요소
- ">"

7) 속성선택자
- 동일한 html 태그의 동일한 속성을 가지고 있는 경우, 안에 들어가있는 속성값에 따라서 스타일을 상이하게 적용한다.
    input[type="text"] {
      background-color: #f00;
    }

    input[type="password"] {
      background-color: #00f;
    }

8) 동위선택자
- 동 : 같을 "동"
- 위 : "위"치
> 같은 위치에 속해있는 요소들간 스타일을 적용하고 싶을때 사용
> 반드시 형제 요소여야 함
▶ 특성 요소의 바로 뒤에 존재하는 형제 찾아올 때 : "+"
▶ 특정 요소의 뒤에 있는 모든 형제 : "~"

9) ★구조선택자 ★
- 일반 구조선택자
ⓐ. 1개의 부모요소 안에 있는 모든 형제들을 형태와 관계없이 입력된 순서대로 번호를 카운팅함
ⓑ. nth-child(n)
     first-child
     last-child
ⓒ. 홀수(2n+1) 혹은 짝수(2n)번째 요소들만 찾아올 수도 있음.
- 형태 구조선택자
ⓐ. 1개의 부모요소 안에 동일한 형태를 가지고 있는 형제들 가운데서 순서대로 번호를 카운팅함
ⓑ. nth-of-type(n)
     first-of-type (동일한 형태의 여러요소중 첫번째)
     last-of-type (동일한 형태의 여러요소중 마지막)

10) 반응선택자
- 사용자가 어떤 액션(*이벤트) 실행 => 반응을 일으킴 (마우스 오버 / 클릭 )
- hover : 마우스 오버
- active : 마우스 클릭

11) 상태선택자
- 사용자의 이벤트 액션을 통해서 특정 태그 및 요소들의 상태가 변화되었다는 것을 감지하고 이를 통해서 스타일을 적용.
- form : 버튼 / checkbox / radio btn

12) 링크선택자
ⓐ. 링크를 가지고 있는 요소들을 선택하기 위한 목적
:link
ⓑ. 특정 링크가 클릭된 경우를 선택하기 위한 목적
:visited

13) ★문자선택자★
- 시작문자선택자
> 특정 문서 안에 시작하는 첫번째 단어 혹은 문장을 선택해서 스타일을 적용하고자 할 때
::first-letter > 시작하는 첫번째 문자열
::first-line > 시작하는 첫번째 행

- 전후문자선택자 (*중요)
> 문서상의 특정 요소의 앞과 뒤에 어떤 스타일링을 적용하고자 할 때 사용할 수 있는 선택자
> 2차원적인 영역만 이야기 하는 것은 X,3차원적인 측면에서도 가능(*특정한 이미지 혹은 아이템을 기준으로 Z축으로 양수 / 음수)
> 스타일 적용할 때, 반드시 content: "";가 들어가야한다.
★ 실무에서 전후문자선택자를 프로페셔널하게 사용하는 방법
> html을 통해서 속성을 정의할때
> data- : 해당 접두어 뒤에 원하는 문자열을 넣으면 고유한 속성이 된다.

* data- 접두어를 활용해서 어디까지 프로그래밍 or 스타일 할 수 있는가?

> position 속성 : 부모 relative / 자식 absolute

> 전후문자선택자를 활용해서 탄생되는 요소는 자식요소

- 반응문자선택자
> 사용자의 이벤트에 반응하는 문자선택자
> 반응선택자 : 문자를 대상 X
> 반응문자선택자 : 문자를 대상
::selection

14) 부정선택자
- 선택한 해당 요소를 제외한 나머지 요소들에게 스타일을 적용하고 싶은 경우 사용
- not( )=> 해당 함수 안에는 부정하고자 하는 기준값이 매개변수로 포함될 수 있다.
